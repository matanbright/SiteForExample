הסבר על התיקיות והקבצים של האתר:
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
* בתיקיה הראשית נמצאים קבצי '.php' שהם הקבצים הראשיים של כל עמוד באתר (אלה קבצים שמפעילים את העמודים ולא נמצא בהם התוכן של כל עמוד).
   (!) הערה: בכל הקבצים הראשיים צריכות להיות אותן שורות קוד שמפעילות את התוכן הקבוע באתר, אך בכל קובץ תהיה בנוסף שורת קוד שמפעילה את העמוד שלו.

* תוכן כל דף נמצא בתיקיה 'PHP - HTML pages content' (בתוך קובץ '.php' שמשלב בתוכו קוד HTML).

* תוכן האתר הקבוע (כגון: כותרת, סרגל כלים, סרגל ימני, כותרת תחתונה) נמצא בתיקיה 'PHP - HTML static content' (בתוך קובץ '.php' שמשלב בתוכו קוד HTML).

* בתוך התיקיה 'PHP' נמצאים קטעי קוד בשפת PHP (שלא משלבים בתוכם HTML).
   (!) הערה 1: קטעי קוד אלה לא אחראיים על עיצוב ומבנה האתר אלא אחראיים על ביצוע פעולות צד שרת (כגון: התחברות לבסיס נתונים, ביצוע התחברות/הרשמה לאתר).

* בתוך התיקיה 'JS' נמצאים קטעי קוד בשפת JavaScript שאחראיים על ביצוע פעולות צד לקוח (כגון: שינוי אלמנטים בדף, פתיחה/סגירה של חלונות).

* בתוך התיקיה 'CSS' נמצאים קבצים האחראיים על העיצוב של האתר.
   (!) הערה: הקבצים מחולקים לקובץ גלובלי, לקבצים המתמקדים כל אחד בחלק קבוע מסויים של האתר (כותרת [header], גוף [main], תחתית [footer]) ולקבצים המתמקדים בכל דף בנפרד.

* בתוך התיקיה 'resources' נמצאים המשאבים של האתר (כגון: תמונות, וידאו, מוסיקה, מסמכים, הרחבות [כגון: Bootstrap]).



הערות:
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
* הקובץ 'htmlHead.php' שבתיקיה 'PHP - HTML static content' הוא הקובץ שמכיל את תחילת קוד ה - HTML של האתר.

* הקובץ 'htmlBody_bottom.php' שבתיקיה 'PHP - HTML static content' הוא הקובץ שמכיל את סוף קוד ה - HTML של האתר.
   הוא יכיל את כל האלמנטים הגלובליים שלא קשורים למבנה הראשי של האתר (כגון: אלמנטים של חלונות קופצים גלובליים).

* הקובץ 'htmlBody_main_toolbars.php' מכיל את תחילת הקוד של אלמנט ה - 'main' של מבנה האתר ואילו סופו של הקוד נמצא בכל אחד מהקבצים של התוכן של כל עמוד.
   (!) הערה: כאשר צריך לשנות את סוף הקוד של אלמנט ה - 'main', צריך לשנות אותו בכל אחד מהקבצים של התוכן של כל עמוד.

* כאשר יוצרים עמוד באתר צריך ליצור בתיקיה הראשית קובץ '.php' (שבו יהיו שורות קוד שמפעילות את העמוד [על פי ההוראות]),
   וצריך ליצור בנוסף קובץ '.php' בתיקיה 'PHP - HTML pages content' (שבו יהיה התוכן של העמוד בשפת HTML [וגם PHP אם רוצים לבצע התניה או בדיקה]).
   (!) הערה 1: צריך גם להוסיף כפתור לעמוד בסרגל הכלים העליון.
   (!) הערה 2: צריך גם לעדכן בסרגל הניווט שנמצא בראש התוכן של כל דף את הפרטים על הדף והדפים שהפנו אליו (כולל קישורים לדפים שהפנו).

* יש לשים סקריפטים של JavaScript בתוך התג 'body' ובסופו (שהם יהיו בשורות הכי אחרונות בתג 'body') ולא בתוך התג 'head'.

* כאשר מתעסקים עם קבצי עוגיה (Cookies) אז יש לשים לב שכאשר משתמשים בשפת PHP בפונקציה 'setcookie()' הטקסט שנשמר בקובץ העוגיה הוא מקודד (זה יגרום לבעיות עם עברית ותווים מיוחדים)
   כאשר מתמשים בשפת PHP כדי לקבל ערך של קובץ עוגייה אז הערך מקודד חזרה, אבל כאשר משתמשים בשפת JavaScript כדי לקבל ערך של קובץ עוגיה אז הערך לא מקודד חזרה וגם כששומרים ערך של קובץ עוגיה הוא לא מקודד.
   לכן כדי לקדד חזרה את הערך יש להשתמש בשפת JavaScript בפונקציה decodeURIComponent() וכששומרים קובץ עוגיה יש לשמור אותו אחרי שהערך קודד באמצעות encodeURIComponent().

* כאשר מציינים נתיב של קובץ בתוך קוד CSS אז צריך לציין את המיקום של הקובץ בייחס לקובץ ה CSS (כלומר, אם הקובץ שרוצים לצרף נמצא בתיקייה אחת לאחור אז צריך לציין זאת על ידי "../").

* אסור לשים רווח בשמות של קבצים ותיקיות! (במקום רווח אפשר לשים קו תחתון).
   (!) הערה: רק בקוד PHP אפשר לציין תיקיה עם רווח בשמה (אבל לא קבצים).

* לא לתת id לאלמנטים פשוטים של טקסט בקוד HTML (כגון: p, h1-6, a). אם רוצים לעצב אותם אז יש לעשות זאת בקוד ה-HTML עצמו ולא בקוד ה-CSS.

* כאשר רוצים ליצור עיצובים מותאמים לניידים יש לבצע התנייה בקובצי ה - CSS, כדי שתהיה הפרדה בין תצוגת האתר במצב שולחן עבודה לבין תצוגת האתר במצב נייד.

* כאשר לא אפשרי להשתמש ב - CSS בלבד כדי ליצור את העיצוב הרצוי לגרסה מותאמת לניידים (למשל כשיש צורך להזיז אלמנטים ממקום למקום לפי הגרסאות) צריך לעשות עותק של האלמנט שרוצים
   לעצב לניידים ולדאוג לכך שבגרסה לשולחן עבודה הוא יוסתר ואילו בגרסה לניידים הוא יוצג והמקור יוסתר.
   (!) הערה: רצוי שלא להגיע למצב שכזה, אך אם אין ברירה ועושים זאת, צריך לדאוג לכך שאם משנים תוכן במקור אז צריך לשנות אותו גם בעותק!

* בסל הקניות שבאתר, לאלמנט של כל מוצר יש ID הכולל את מספר המוצר כדי להבדיל בין האלמנטים הזהים של כל מוצר בקוד ה - JavaScript כאשר מבצעים הזמנה.
   לדוגמא: האלמנט 'price1' הוא אלמנט המציין מחיר של מוצר מספר 1.
